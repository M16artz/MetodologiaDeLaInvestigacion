# 📊 Trazabilidad de Investigación: Redes Bayesianas en Diagnóstico Médico

Este repositorio contiene la bitácora, documentación y progreso del proyecto de investigación para la asignatura de **Metodología de la Investigación** (Cuarto Ciclo, Computación - Universidad Nacional de Loja).

## 🎯 Pregunta de Investigación
> **¿Cómo se utilizan las redes bayesianas para modelar la incertidumbre en sistemas de diagnóstico médico?**

## 🧠 Marco Conceptual Básico
El proyecto se fundamenta en la aplicación del Teorema de Bayes para actualizar la probabilidad de una hipótesis médica (como una enfermedad) basada en nueva evidencia (como un síntoma o resultado de laboratorio):

$$P(H|E) = \frac{P(E|H) \cdot P(H)}{P(E)}$$

Donde $P(H|E)$ representa la probabilidad de la enfermedad dada la evidencia observada, un pilar fundamental para construir los grafos acíclicos dirigidos (DAGs) que conforman las redes bayesianas.
